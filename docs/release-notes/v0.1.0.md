# SpecExplorerKit v0.1.0

The first release of **SpecExplorerKit (SEK)** — a modern, CLI-first,
cross-platform revival of Microsoft Spec Explorer, powered by the Z3 theorem
prover and the Cord scenario language, and packaged as a **Spec Kit community
extension**.

## Highlights

- **`sek` CLI** — `init`, `validate`, `explore`, `view`, `test` on .NET 8, no Visual Studio required.
- **Z3-powered parameter generation** — `Condition.In`, `Condition.IsTrue`, and combination strategies (interaction, pairwise) are solved by Z3.
- **Cord language** — the full behavior algebra (`; | || ||| |?| -> & * + ? {n} _ ... !`) plus configurations, parameter domains, and `construct` machines.
- **Reachable-object domains** — reference-typed rule parameters draw from the live objects in the current state.
- **Conformance** — replay an exploration against a real implementation and get a pass/fail report.
- **Spec Kit extension** — `/speckit.sek.model`, `/speckit.sek.explore`, `/speckit.sek.verify`.

## Validated on the classic sample suite

All nine Spec Explorer 2010 samples were ported to the modern toolchain and are
explored successfully by this release (see `samples/`), alongside a TPC-C model
that passes full conformance replay (2,446 states / 12,706 transitions).

## Assets

| Asset | Purpose |
|---|---|
| `spec-kit-sek.zip` | The Spec Kit extension (install with `specify extension add`). |
| `SpecExplorerKit.Tool.0.1.0.nupkg` | The `sek` .NET global tool. |

## Install

```bash
# The sek tool
dotnet tool install -g SpecExplorerKit.Tool --version 0.1.0 --add-source <folder-with-nupkg>

# The Spec Kit extension
specify extension add spec-kit-sek --from https://github.com/stuartpa_microsoft/sek/releases/download/v0.1.0/spec-kit-sek.zip
```

See the [documentation](https://github.com/stuartpa_microsoft/sek/tree/main/docs)
for the full quickstart, concepts, and reference.
