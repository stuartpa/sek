# SpecExplorerKit v0.1.1

Cross-platform Z3, out of the box.

## Fixed

- **Linux Z3 support.** The `Microsoft.Z3` package ships native libraries only for
  Windows and macOS, so `sek` previously could not run Z3-backed parameter generation
  on Linux (it fell back to an incompatible system `libz3`). The `sek` tool now
  **bundles a matching `libz3.so`**, so exploration and parameter generation work on
  **Windows, macOS, and Linux** with no extra setup.
- CI now runs the release pipeline on `ubuntu-latest` and validates all samples with
  real Z3 on Linux.

## Assets

| Asset | Purpose |
|---|---|
| `spec-kit-sek.zip` | The Spec Kit extension (install with `specify extension add`). |
| `SpecExplorerKit.Tool.0.1.1.nupkg` | The `sek` .NET global tool (Windows/macOS/Linux). |

## Install

```bash
dotnet tool install -g SpecExplorerKit.Tool --version 0.1.1 --add-source <folder-with-nupkg>
sek z3   # prints SATISFIABLE on all platforms
```

See the [documentation](https://github.com/stuartpa/sek/tree/main/docs) for the full
quickstart, concepts, and reference.
