This sample demonstrates:

- Modeling of a simple component including combinatorial parameter generation and exploration

How to run the sample:

1.	You must run VS with administrator privileges otherwise all test cases will fail.
2.  Try exploring machines in config. Look at the way how parameter
    combinations are declared for Explore.   
3.	The project ModeledTests contains test cases generated from the model. To re-generate the test suite 
    from the model, open Exploration Manager, select machine TestSuite, and click "Generate Test Code". 
4.	All the test result can be observed in Test Results window.


Known issues:

1.	All the test cases fail on Windows XP.
